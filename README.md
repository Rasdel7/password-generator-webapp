# Password Generator & Strength Checker 🔐

Generate uncrackable passwords and analyze
the strength of any password with entropy calculation.

## Live Demo
[Click here](https://password-generator-webapp-cygzb9xyeygxnr4xgmtlyn.streamlit.app)

## Features
- Generate 1-10 passwords at once
- Customizable length (8-64 characters)
- Real entropy calculation in bits
- Crack time estimation (trillions of guesses/sec)
- Full security checklist
- Memorable passphrase generator
- Password security tips and attack methods

## Tools Used
- Python, Streamlit, Pandas, Matplotlib

## How to Run Locally
pip install streamlit pandas matplotlib
streamlit run app.py
